## 두더지 잡기 APP ##

**두더지잡기** 앱은 *다양한 이미지*, *음악*을 사용한 안드로이드 버전의 게임입니다.

- 간단한 SQLite를 이용한 점수, 이름 기록
- 터치와 터치뗏을때, 각각의 다른 반응
- 점수와 시간이 일정 지점 도달할 경우, 음악 변경
- 간단한 소스를 통한 Time killing Game 개발 가능
